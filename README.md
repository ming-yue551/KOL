基于节点表示学习的社媒空间意见领袖识别算法
MDCE-GAT：多模态解耦与对比学习增强的图注意力网络
================================================================================

PROJECT INTRODUCTION
--------------------------------------------------------------------------------
本项目实现社交媒体环境下意见领袖自动识别算法，通过融合用户网络结构、
交互行为与内容语义特征，构建高精度、高鲁棒性的识别模型。
支持两类典型社媒话题场景：
  1. 情感共鸣型话题（Dream数据集）
  2. 价值观驱动型话题（Hair数据集）

ENVIRONMENT REQUIREMENTS
--------------------------------------------------------------------------------
python >= 3.8
torch
torch_geometric
pandas
numpy
scikit-learn
matplotlib

EXPERIMENT RESULTS
--------------------------------------------------------------------------------
Dream话题（情感驱动场景）
  Top-5 准确率：0.90
  NDCG@10：0.913
  影响力特征：头部高度集中，网络结构主导

Hair话题（内容驱动场景）
  Top-5 准确率：0.90
  NDCG@10：0.897
  影响力特征：多中心分布，内容质量主导

MODEL FEATURES
--------------------------------------------------------------------------------
1. 多模态特征解耦与融合，解决异质特征表征冲突
2. 对比学习增强模块，提升模型抗噪声鲁棒性
3. 自适应平衡结构特征与内容特征权重
4. 双场景泛化能力强，适配不同类型社媒话题
5. 可解释性强，可直接落地舆情治理、内容运营等场景
